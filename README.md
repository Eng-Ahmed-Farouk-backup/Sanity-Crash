Your choices are not just decisions... they are consequences.

a Comic Horror Visual Novel , can be played in CLI 

Search for the Light , in the Night where they Hide to be Right.

## Story
You go to the cemeteries in the middle of the Desert with your Family , You took a nap while they finish. you wake up to find Night came out and your family Left you alone.

## Features

- Branching story-telling system
- Choice-based gameplay with real consequences
- Comic horror (Cringe Horror Story)

## Architecture
### CLI
- `interactive_consol.py` -> Contains Helper Functions for Desplaying Content in CLI
- `items.py` -> Contains Items Used in game
- `main.py` -> Contains the main menu and opearating to choose between CLI and GUI
- `enemy.py` -> Defines the Enemy class with stats, combat, and loot drop methods
- `player.py` ->  Defines the Player class with stats, inventory, and combat methods 
### Demo
- `demo.py` -> contains all other files and is used to make the demo
